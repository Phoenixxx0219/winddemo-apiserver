# 系统后端服务器

## API使用说明
1. 获取真实图片数据
```
/realimage/{date}/{type}/real/{filename}
```
2. 获取预测图片数据
```
/forcastimage/{date}/{type}/forcast/{time}/{filename}
```
3. 获取单体轮廓数据
```
/api/convective/tracking
请求参数：
{
  "time": "2024-06-04 00:00",
  "interval": "6",
  "algorithm": "forcast"
}
```

## 源代码使用说明
启动后端服务器：`python apiServer.py`
